    Logger

    new Logger(logger, tagsopt, contextopt, categoryopt)

    This class contains a (tags, context, category) tuple. Its methods accept tags, context, and category, which, if provided, are combined with the object's corresponding properties. For example, if the object is created with tags = ['apple'] log('banana') will use the tags 'apple' and 'banana.'
